React

React一个用于构建用户界面的JavaScript库

##安装bower
bower 前端框架器 jQuery angular bootstrap
bower 是一个全局的命令行工具，需要通过npm 全局安装

npm 是安装node的时候自动安装
npm node包管理器

##安装react
$ bower install react babel –save

react是需要安装的模块，babel相当于翻译，因为浏览器不能识别react代码，所以通过babel帮助翻译，把react原生代码翻译可识别的JS代码

bower install 之后会在当前项目下增加一个bower_components的文件夹，里面放着安装的模块

react.js是React的核心库
react-dom.js是提供与DOM相关的功能，会在window下增加ReactDOM属性
brower.js的作用是将JSX语法转为JavaScript语法
script的type属性为text/babel,因为React独有的JSX语法，跟JavaScript不兼容

要把bower_component添加到git的忽略列表里面,只需要把依赖的模块告诉别人就可以，别人可以自行安装，还有就是包太大。

##ReactDOM.render()

引入react-dom.js之后会在window下增加一个ReactDOM属性，也就是说会增加这样一个全局变量

参数1 是要渲染的内容 也就是要插入到DOM元素内部的标签,有且只能有一个顶级元素
参数2 画布 ，要渲染到那个DOM元素的内容

如果想在htmldiamante里面内部写JS代码，那么需要套一对大括号，{ } 里面代表的代表是JS代码，一般会经过运行会返回一个东西，会被渲染出来。

##createClass
通过createClass可以创建一个自定义组件,这个自动以组件用法和普通的HTML标签一样用，组件名首字母必须大写，必须有结束标签，
属性一般是有父组件使用此子组件的时候传入的
this代表当前组件的实例

##优点

  • 方便引用，精简代码
  • 方便修改，如果要修改只要改一处就可以

getInitialState获取初始状态, 返回一个状态对象，这个状态对象的属性和值自己决定
this.state 获取返回值

##react元素
React 当中的元素事实上是普通的对象，所以元素属性不能随便写。React DOM 可以确保 浏览器 DOM 的数据内容与 React 元素保持一致。

渲染有差异的，有变化的。

##属性/状态

  • 属性具有不变性
    属性是由外部传入的，而且不会改变
  • 状态
    是由内部初始化，并且随着时间的推移，随着外界的影响会发生变化
